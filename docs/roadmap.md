# ERP Bibit Net Roadmap

> 🌱 **ERP Bibit Net**
> Enterprise Resource Planning berbasis **Next.js 16**, **React 19**, **Prisma ORM 7**, dan **PostgreSQL 16**.
>
> Architecture:
>
> * Multi Company
> * Multi Branch
> * Multi Warehouse
> * Modular Architecture
> * Domain Driven Design
> * Monolithic Next.js

# Development Principle

ERP Bibit Net dibangun berdasarkan prinsip berikut.

```text
Foundation

↓

Architecture

↓

Convention

↓

Domain

↓

Module

↓

Feature

↓

User Interface
```

Business Domain selalu menjadi dasar sebelum implementasi UI maupun database.
